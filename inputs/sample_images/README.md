# Sample product images

Drop reference product images here. The paths in `inputs/products.json` point to this folder by default.

For the seed products you'll want:

- `shapewear.jpg` — Slim Fit Shapewear Bodysuit
- `earbuds.jpg` — Wireless Earbuds Pro
- `serum.jpg` — Hydrating Vitamin C Serum

`--dry-run` will warn (exit 1) if these files are missing but still print the prompts. Live runs require the files to exist.
